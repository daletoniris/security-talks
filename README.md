## About Chubut Hack

Chubut Hack was a cybersecurity meetup community (2021-2022) that hosted 20+ talks covering topics from offensive security, wardriving, and adversarial AI to Linux administration and web security. Speakers included professionals from Argentina, Colombia, and other LATAM countries.
